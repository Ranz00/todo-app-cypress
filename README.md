## Description
A simple web application to manage tasks with categories and localStorage persistence. Includes automated UI tests using Cypress.

## Contents
- `index.html`: Application interface.
- `style.css`: Basic styling.
- `script.js`: Logic for adding, deleting, and persisting tasks.
- `cypress/integration/todo.spec.js`: Automated tests covering edge cases.

## How to Run
1. Open `index.html` in a browser.
2. Install Cypress (`npm install cypress`).
3. Run `npx cypress open` and execute the tests.

## Skills Demonstrated
- Frontend development with HTML, CSS, and JavaScript.
- Data persistence using localStorage.
- Automated UI testing with Cypress.
- Validation and error handling.
